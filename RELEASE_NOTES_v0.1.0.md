# pickbot v0.1.0

## Summary

The first public release of pickbot.

## Highlights

- Config-driven foreground keyboard and mouse automation
- `flow.txt` for the main workflow
- `flow_test.txt` for quick action validation
- Supported actions:
  - `click`
  - `hold`
  - `drag`
  - `key`
  - `wait`
  - `wait_pixel`
  - `wait_image`
- Hotkeys:
  - `F7` load test flow
  - `F8` start or stop
  - `F9` reload main flow
  - `F10` exit

## Release files

- `pickbot.exe`
- `config.json`
- `flow.txt`
- `USER_GUIDE.md`
- `README.md`
- `NOTICE.txt`
- `VERSION.txt`
- `SHA256SUMS.txt`

## Notes

- This is foreground automation and will take over keyboard and mouse input while running.
- This release uses simulated input and screen-state detection.
- This release does not directly read or write the target process memory.
- Use only the official release channel.
- If you paid for this tool through an unofficial seller, request a refund.
